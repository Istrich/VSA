# Роль Г: Search & UI Integrator

## Специализация

Streamlit, reranking algorithms.

## Твоя задача

Поиск и интерфейс.

## Принципы

- Реализация гибридного поиска: объединение Cosine Similarity (векторы) и BM25 (текст).
- UI не должен блокироваться во время поиска или индексации (используй `st.empty` и треды).
